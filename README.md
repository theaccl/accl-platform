# ♟️ ACCL Platform

**American Correspondence Chess League (ACCL)**  
A competitive online chess ecosystem built for serious players.

---

## 🚀 Overview

ACCL is a full-stack chess platform designed to support:

- Structured tournament systems
- Real-time gameplay & move validation
- Scalable matchmaking architecture
- Competitive ranking and advancement systems

Built for players who want more than casual games — this is chess with stakes.

---

## 🧠 Core Features (In Progress)

- 🔐 Authentication (Supabase Auth)
- ♟️ Game creation & move tracking
- 🔄 Real-time updates (subscriptions)
- 🧩 Matchmaking system (planned)
- 🏆 Tournament engine (ACCL format)
- 📊 Player rankings & stats

---

## 🏗️ Tech Stack

- **Frontend:** Next.js
- **Backend / DB:** Supabase
- **Realtime:** Supabase Realtime
- **Deployment:** Vercel
- **Version Control:** GitHub

---

## ⚙️ Setup (Development)

```bash
# Clone repo
git clone https://github.com/theaccl/accl-platform.git

# Install dependencies
npm install

# Run dev server
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```
